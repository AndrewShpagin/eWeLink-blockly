Using scripts generated by the  IOT-PROXY.COM
  1. First, provide basic eWeLink access information - email, password, region using the Login button. We store that strings locally and never store it anywhere except your computer.
  2. Use blocks to create scenario to be executed periodically, for example once in 5 or 10 minutes. For example, check temperature or state of the devices and then do some actions - switch devices, collect information into the Google Sheets or anything else.
  3. Create a new blank Spreadsheet (https://docs.google.com/spreadsheets/u/0/)
  4. Go to Tools->Script editor
  5. Paste the code to the Script editor text input area.
  6. Press the + right to the Libraries
  7. Copy and paste the code, press Look up, and then Add

  8. Now run the script with |> icon. Check if all works as expected.
  9. Go to triggers
  10. Add trigger
  11. Setup and add the time-driven trigger
  12. Now your script will be executed every 5 minutes (set another time period if need).
  13. It is not mandatory to be programmer to use this App, but if you are familiar with programming, look the API. Each function from the list may be used as ew.functionCall(...). If you are programmer, start to play with blocks, then just write a code using the API.
